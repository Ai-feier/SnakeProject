# SnakeProject

<u>入门级项目，适合新手入门，代码相对简单，但逻辑相对清晰，适合学习。</u>



## 1、主要类

```
	Node类：节点，用于构造食物和蛇的基础类
	SnakeControl类：蛇的控制类，主要控制类
	SnakeFrame类：绘制游戏窗口类
```
